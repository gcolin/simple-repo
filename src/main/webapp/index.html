Simple repo

A tiny Maven proxy and repository.

All repositories

Configuration

The configuration interface is accessible through JMX. If you want an easy access to the JMX interface use the project jmx-web-console. Another way to configure Simple Repo is to edit the config.xml it self located is ~/.simplerepo.

The repository

There are 3 type of repositories:

  • The proxy repositories with a remote URL
  • The group repositories with included repositories
  • The hosted repositories with no remote URL and no included repository

The repositories are stored in ~/.simplerepo.

The Max Snapshots value

The number maximum of snapshot version in a repository. If the number is exceeded the oldest snapshot will be removed. A value of 0 means unlimited number of snapshots.
